# Organizational-Hierarchy-Dashboard
This Power BI report provides a complete, interactive visualization of an organization’s structure, employee distribution, and Line‑of‑Business (LOB) segmentation.
It includes dynamic filters, KPI highlights, and an expandable org‑chart view that helps users explore reporting relationships and workforce composition with ease.
             1.KPI Summary Cards for Headcount, Managers, Production Staff, IT Support, and Selected Count
             2.Dynamic Filters (LOB, Name, Supervisor, Role) with one‑click "Clear All"
             3.LOB Distribution Visuals including bar charts and donut charts
             4.Search Functionality to quickly locate employees within the hierarchy
